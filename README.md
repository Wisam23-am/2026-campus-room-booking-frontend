# Campus Room Booking System - Frontend

React + TypeScript frontend untuk sistem peminjaman ruangan kampus.

## Prerequisites
- Node.js 18+ 
- npm atau yarn

## Setup
1. Clone repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` dan sesuaikan konfigurasi
4. Jalankan aplikasi: `npm start`

## Tech Stack
- React 18
- TypeScript
- Axios
- React Router

## Development
Branch: `develop` (default)
